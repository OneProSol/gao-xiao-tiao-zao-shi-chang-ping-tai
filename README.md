### 一、相关技术
- 前端：`thymeleaf`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示

系统分为：用户、管理员。

用户功能：1. 登录、2. 注册、3. 热门闲置商品、4. 商品详情、5. 我的订单、6. 购物车

管理员功能：1. 登录、2. 用户管理、3. 分类管理、4. 订单管理、5. 闲置商品管理
